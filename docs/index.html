Privacy Policy

Last updated: 1 July 2026

Developer

BellApp is developed and published by Ilja Grim.

Privacy contact

bellappsupport@gmail.com

Summary

BellApp does not collect, transmit, sell, rent, or share personal or sensitive user data. BellApp does not use advertising, analytics, tracking, user accounts, location, contacts, camera, microphone, or remote servers.

Data collection and sharing

BellApp does not collect or share personal data, device identifiers, usage analytics, diagnostic data, location data, contact information, financial information, health information, messages, media, or files.

Local app settings

BellApp stores only the selected chime sound, chime interval, run mode, and ON/OFF state. These settings remain on the user's device and are not transmitted to the developer or any third party.

Permissions

Notifications: used only to display BellApp status and controls.

Alarms and reminders: used only to schedule user-selected chimes at precise clock times.

Receive boot completed: used only to restore an enabled schedule after the device restarts.

External links

The GitHub button may open the project page in the user's chosen browser. BellApp itself does not request Internet permission and does not receive information about the user's browsing.

Data security

Because BellApp does not transmit or remotely store personal data, no personal user data is sent over a network or held on a developer-controlled server.

Retention and deletion

Local settings remain until the user clears BellApp storage or uninstalls the app. BellApp has no account system and no server-side user profile.

Children

BellApp does not knowingly collect data from children or from any other users.

Changes to this policy

Material changes will be reflected in the privacy policy available inside BellApp and on this page.

Contact

Questions about this privacy policy can be sent to bellappsupport@gmail.com.
